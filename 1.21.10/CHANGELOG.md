# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [v21.10.1-1.21.10] - 2025-10-24

### Added

- Add shelf variants for all wood types thanks to the
  amazing [leftchaotix](https://www.curseforge.com/members/leftchaotix)

### Changed

- Wooden blocks now catch on fire from nearby fire sources
- Brown mooshrooms now also have a fancy updated texture like their red counterparts

### Fixed

- Fix stripped stems and signs not being usable as furnace fuel
- Fix hanging signs being crafted with wooden planks instead of stripped stems
- Fix chest boat item textures being placed one pixel too low

## [v21.10.0-1.21.10] - 2025-10-10

### Changed

- Update to Minecraft 1.21.10
